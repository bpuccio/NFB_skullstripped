The Neurofeedback Skull-stripped (NFBS) repository is a database of 125 T1-weighted anatomical MRI scans that are manually skull-stripped. NFBS allows researchers and data scientists to test brain extraction methods and have training data to be used in machine learning based algorithms. The data was collected as a part of the Enhanced Rockland Sample Neurofeedback Study. NFBS is available for download on the NFB_skullstripped Github page.

For more information, please see:

Puccio et al. A repository of skull-stripped T1-weighted structural MRI data. (2016).

Abstract

Brain extraction, also known as skull stripping, is one of the first steps used in the preprocessing of neuroimaging data. The purpose is to remove non-brain tissue from structural magnetic resonance imaging (MRI) data in order to increase the quality of subsequent analyses. Many algorithms have been developed in order to solve the problem, though none have been able to overcome all of the challenges. In order to facilitate the development of automatic brain extraction methods, 125 T1-weighted anatomical MRI scans from The Enhanced Rockland Sample Neurofeedback Study were used to create a repository of manually skull stripped T1-weighted MR images.

The BEaST method1 (brain extraction based on nonlocal segmentation technique) was used to create the initial masks. A few masks were chosen to be manually edited and used as the reference library when running BEaST again. Manual editing was performed using the Freeview visualization software from Freeview. In total, 85 of the masks were hand edited and 40 were deemed good enough to not be edited. In order to compare the quality of the repository, a few skull stripping algorithms were used on the anatomical scans. The results were quantified using our repository as gold truth data and comparing with masks generated from other skull stripping algorithms (BET2, 3dSkullStrip3, Freesurfer HWA4).

Contents of the Repository

The repository contains data from 125 participants, ranging from 21 to 45 years old. For each participant, the repository contains:

  • Structural T1-weighted anonymized (de-faced) image
  • Skull-stripped image
  • Brain mask

The structural MRI data was collected as a part of the Neurofeedback study in the Enhanced Nathan Kline Institute - Rockland Sample. The resolution of the images is 1mm3 and each file is in NiFTI format (.nii.gz).

Downloading the Repository

Please go to our github page for download. The total amount of free hard disk space required is 1.8 GB.

The images are available in NiFTI format (.nii.gz).
